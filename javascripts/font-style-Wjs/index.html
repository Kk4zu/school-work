Change Font style with javascript

Attention! All changes happen with javascript. List font style are changed pushin the button. List supposition font style is "Courier New" font size is 16px.

But what happens when you press the button 10 times?

Change the font style of this list by pressing the button.

  1. First press of the button changes font style to "Roboto"
  2. Second press of the button changes font style to "Manslava"
  3. Third press of the button changes font style to "Grenze"
  4. Fourth press of the button changes font style to "Turret Road"
  5. Fifth press of the button changes font style to "Courier New"
Font style now: "Courier New"
